== Description ==

myGDEX Prime for WooCommerce is myGDEX Prime's official extension for WooCommerce on WordPress. Manage your domestic and international shipments easily.

FEATURES
- Fast and easy consignment note creation of your domestic and international orders.
- Bulk Consignment Note Creation allows you to create consignment notes for multiple orders.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Under WooCommerce - Setting - Shipping - myGDEX Prime, key in your user token to use the plugin.


== Frequently Asked Questions ==

= Can I Use This Plugin Without Being A myGDEX Prime Member? =

No. You need to be a valid myGDEX Prime member to use this plugin. You can register to be a myGDEX Prime member at https://myprime.gdexpress.com/
